# Riddhi Enterprises Solar Website

This is a static HTML website created from the PDF brochure of Riddhi Enterprises, a solar energy company.

## Features

- Responsive design using Bootstrap 5
- Hero section with gradient background
- Navigation bar
- Card-based layouts for services and work
- Collapsible work list
- Placeholder images (replace with actual photos)
- Contact information

## How to View Locally

Open `index.html` in your web browser.

## Deployment to GitHub Pages (Free Hosting)

1. Install Git from https://git-scm.com/download/win
2. Open a terminal in this folder
3. Run the following commands:

   ```
   git init
   git add .
   git commit -m "Initial commit"
   ```

4. Create a new repository on GitHub (github.com)
5. Copy the repository URL (e.g., https://github.com/yourusername/solar-website.git)
6. Run:

   ```
   git remote add origin https://github.com/yourusername/solar-website.git
   git push -u origin main
   ```

7. Go to your repository on GitHub
8. Click Settings > Pages
9. Under Source, select "Deploy from a branch"
10. Select "main" branch and "/ (root)" folder
11. Click Save
12. Your site will be live at https://yourusername.github.io/solar-website/
